java的三个版本
  • java SE:java基础,桌面应用程序
  • java ME:智能机出现之前用于手机游戏开发,但现在应该没人用了
  • java EE:企业应用开发,javaweb一类的,就是在java se基础上封装了一些框架.
jdk jvm jre 区别

jdk(Java Development Kit)是java的开发和运行环境
jre(Java Runtime Environment)是java的编译和运行环境
jvm就是java虚拟机,运行.class 的核心部分

java安装环境变量的配置

JAVA_HOME=%安装路径%\Java\jdk path=%JAVA_HOME%\bin
系统默认先去当前路径下找要执行的程序，如果没有，再去path中设置的路径下找。

classpath:下面一段话引用自参考2

注意：在定义classpath环境变量时，需要注意的情况
如果没有定义环境变量classpath，java启动jvm后，会在当前目录下查找要运行的类文件；
如果指定了classpath，那么会在指定的目录下查找要运行的类文件。
还会在当前目录找吗？两种情况：
1）：如果classpath的值结尾处有分号，在具体路径中没有找到运行的类，会默认在当前目录再找一次。
一般不指定分号，如果没有在指定目录下找到要运行的类文件，就报错，这样可以调试程序。
2）：如果classpath的值结果出没有分号，在具体的路径中没有找到运行的类，不会再当前目录找。

javac java

java:启动jvm,加载运行需要的类库,执行.class文件
main函数是程序执行的初始入口.

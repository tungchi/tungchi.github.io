关于文件上传大小限制的一次问题记录

问题:用公网ip访问上传文件会报403错误.

解决思路:
开始的时候,就从网上查阅资料,修改nginx.conf配置,但是外网还是不能上传文件.
后来想了下用内网默认80端口上传文件,发现可以上传.(这里应该就可以看出应该不是本机服务器的nginx配置原因)
但是因为不自信,还是查阅网上各种方法去修改实验,还是不行.浪费了一天时间.
最后在日报中向领导报告这个问题.
老大认为可能是别的服务器的转发的时候作的限制,后来发现果然是这个问题.
